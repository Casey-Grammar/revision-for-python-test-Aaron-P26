# Python Revision Exercises

## Overview
This repository contains a series of Python revision exercises designed to help you practise core programming concepts.

Each exercise is provided in a separate **TaskX.py** file.

---

## Your Task
For this assignment, you are required to:

- Open **each of the `TaskX.py` files**
- Complete or fix the existing code so that the program correctly solves the exercise
- Ensure the program runs without errors and produces the correct output

Some tasks may contain incomplete code, while others may contain errors that need to be identified and corrected.

---

## Sample Solutions
For each task, a **sample solution** has been provided.

- You may refer to the sample solution **if you are unsure how to proceed**
- Try to solve the task **on your own first** before looking at the solution
- The sample solution is intended to support your learning, not replace your own attempt

---

## How to Work on the Tasks
1. Open this repository in **GitHub Codespaces**
2. Open one `TaskX.py` file at a time
3. Read the comments and instructions in the file carefully
4. Modify the code as required
5. Run the file in the terminal to test your solution:
   ```bash
   python TaskX.py
